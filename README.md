# FastScreenshot

## Introduction
Console application for take screenshot and save it local

## Coding

### Prerequisites
- git
- dotnet core

### Prepare
```shell
$ git clone https://github.com/samuele-cozzi/FastScreenshot.git
$ code .\FastScreenshot\
```

```shell
$ dotnet restore source
```

### Execute

```shell
$ dotnet build source
$ cd .\source\
$ dotnet run
```

### Delivery
```shell
$ dotnet publish
```
